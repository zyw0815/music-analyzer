# Changelog

## [2.0.1] - 2026-07-03

### Fixed
- 修复 DSF/DFF 分析时 ffmpeg DSD 转 PCM 失败的问题。
- 修复 DSF/DFF 音频信息误显示为临时 WAV 参数的问题。
- 修复 DFF 采样率、声道、DSD Rate、码率和质量检测码率评分显示异常的问题。

## [2.0.0] - 2026-07-01

### Added
- 内置播放器
- 质量检测（7 维度评分）
- 频谱分析（频谱图、热力图、频率分布）
- 波形显示（可缩放、RMS 包络、削波/静音标记）
- 声道分析（L/R 频谱、相位相关、Mid/Side）
- 音频信息（完整元数据展示）
- DSD 格式支持（DSF/DFF）
- 拖拽上传
- 暗色主题
- GitHub Actions CI/CD
